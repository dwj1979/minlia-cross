### Minlia Cross 使用NGROK提供外部域名访问

### 使用方式,已发布到MAVEN中央库, 在当前项目中添加如下依赖即可

```
    <dependency>
      <groupId>com.minlia.cross</groupId>
      <artifactId>minlia-cross</artifactId>
      <version>1.0.2.RELEASE</version>
    </dependency>
```
